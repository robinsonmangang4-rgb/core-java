✅ Basic Level ( 1-15 )

1️⃣ Print numbers from 1 to 10 

for (int i = 1; i <= 10; i++){
    System.out.println(i);
}

Explanation : Loop starts at and run until 10 inclusive.

2️⃣ Print numbers from 10 to 1

for (int i = 10; i >= 1; i--){
    System.out.println(i);
}

Explanation : Decrement loop to reverse the order.
