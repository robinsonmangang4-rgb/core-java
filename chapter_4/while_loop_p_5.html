✅ 1️⃣ What is a While Loop?

A while loop is used to execute a block of code repeatedly as long as a condition is true.

⇒ It is also called an entry-controlled loop because the condition is checked before execution.

2️⃣. Syntax of While Loop 

while ( condition) {
    // code to be executed
}

How it works :

  • Condition is checked.
  • If true → execute block
  • After execution → condition checked again.
  • If false → loop stops.

3️⃣. Basic Example 

public class WhileExample {
    public static void main(String[] args) {
        int i = 1;

        while (i <= 5) {
            System.out.println(i);
            i++;
        }
    }
}

Output: 

1
2
3
4
5

Explanation:

  • Start with i = 1
  • Loop runs while i <= 5
  • i++ increases value each time
  • Stops when i = 6

4️⃣. Infinite While Loop 

while ( true ) {
    System.out.println("Hello World!")
}

→ This runs forever because condition is always true.

5️⃣ While Loop with User Input. 

import java.util.scanner;

public class WhileInput{
    public static void main(string [] args ){
        Scanner sc = new Scanner (System.in);
        int num;
        while ( true){
            System.out.print("Enter number (0 to stop) : ");
            num =sc.nextInt();

            if( num == 0 ){
                break;
            }

            System.out.println(" You entered: " + num);

        }
    }
}

Explanation:

  • Loop runs infinitely
  • Stop when user enter 0.
  • break exits loop.

6️⃣ While Loop vs For Loop.

Feature While Loop For Loop
Use case Unknown iterations Known iterations
Structure Simple Compact
Initialization Outside loop Inside loop

7️⃣. Do-While Loop (Important)

→ Special type of while loop that executes at least once. 

int i = 1;
    do {
        System.out.println();
        i++;
} while (i <= 5);

Difference :

  • while → checks first.
  • do-while → executes first.

8️⃣ Nested while Loop 

public class NestedWhile {
    public static void main(String[] args) {
        int i = 1;

        while (i <= 3) {
            int j = 1;

            while (j <= 2) {
                System.out.println("i = " + i + ", j = " + j);
                j++;
            }

            i++;
        }
    }
}

9️⃣ Common Mistakes.

❌ Missing increment → Infinite loop. 

int i = 1;
while (i <= 5) {
    System.out.println(i);  // i never changes
}

❌ Wrong condition.


while (i >= 5) // may never run

1️⃣1️⃣Practice Programs (Important )

1. Print 1 to 10 

int i = 1;
while (i <= 10) {
    System.out.println(i);
    i++;
}

2. Sum of first N numbers 

int i = 1, sum = 0;

while (i <= 5) {
    sum += i;
    i++;
}

System.out.println("Sum = " + sum);

3. Reverse a Number 

int num = 1234;
int rev = 0;

while (num != 0) {
    int digit = num % 10;
    rev = rev * 10 + digit;
    num = num / 10;
}

System.out.println("Reverse = " + rev);

4. Check Palindrome

int num = 121, temp = num, rev = 0;

while (num != 0) {
    int digit = num % 10;
    rev = rev * 10 + digit;
    num = num / 10;
}

if (temp == rev)
    System.out.println("Palindrome");
else
    System.out.println("Not Palindrome");

5. Factorial 

int n = 5;
int fact = 1;

while (n > 0) {
    fact *= n;
    n--;
}

System.out.println("Factorial = " + fact);
