Introduction to Java & Basics

1️⃣ What is Java?

Java is a programming language used to create desktop applications, web applications, mobile apps (Android), and enterprise software.

3️⃣ History of Java

  • Developed by James Gosling
  • Released In 1995
  • Originally by Sun Microsystems
  • Now owend by Oracle

3️⃣ Features of Java

  • Simple & Easy to Learn
  • Object Oriented
  • Platform Independent
  • Secure
  • Robust & Scalable

4️⃣ Platform Independent

Java follows the principle: "Write Once, Rune Anywhere"

Java program run on any operating system because of the JVM.

5️⃣ Java Architecture 

                .java file
                    ↓
                Java Compiler
                    ↓
                .class file(Bytecode)
                    ↓
                JVM (Java Virtual Machine)
                    ↓
                  Output

6️⃣ JDK JRE & JVM

  • JVM: Executes Java Bytecode
  • JRE: JVM + Libraries
  • JDK: JRE + Compiler + Tools

Relation: JDK → JRE → JVM

7️⃣ First Java Program

             class Hello
              {
                public static void main(String[] args)
                 {
                  System.out.println("Hello World");
                 }      
              }

8️⃣ Program Explanation

  • class Hello → Class Name
  • main() → Program execution starts here
  • System.out.println() → Prints output
