1️⃣ What is an else-if ladder?

An else-if ladder is used when we want to check multiple conditions one by one. Java executes the first condition that becomes true and skips the rest.

2️⃣ Why do we need else-if ladder?

Used when multiple decisions are required, like grading systems or validations.

3️⃣ Syntax

if (condition1) {
    // code
} else if (condition2) {
    // code
} else if (condition3) {
    // code
} else {
    // default code
}

4️⃣ Basic Example

int num = 0;

if (num > 0) {
    System.out.println("Positive number");
} else if (num < 0) {
    System.out.println("Negative number");
} else {
    System.out.println("Zero");
}

5️⃣ Student Marks Example

int marks = 85;

if (marks >= 90) {
    System.out.println("Grade A");
} else if (marks >= 80) {
    System.out.println("Grade B");
} else if (marks >= 70) {
    System.out.println("Grade C");
} else if (marks >= 60) {
    System.out.println("Grade D");
} else {
    System.out.println("Fail");
}

6️⃣ Flowchart Thinking

Conditions are checked from top to bottom. When one condition is true, Java executes that block and exits the ladder.

7️⃣ Common Mistakes

Writing conditions in the wrong order leads to incorrect results. Always check higher conditions first.

8️⃣ else-if vs multiple if

  • Multiple if → multiple blocks can execute
  • Else-if ladder → only one block executes

9️⃣ Advanced Login Example

String username = "admin";
String password = "1234";

if (username.equals("admin") && password.equals("1234")) {
    System.out.println("Login successful");
} else if (!username.equals("admin")) {
    System.out.println("Invalid username");
} else if (!password.equals("1234")) {
    System.out.println("Invalid password");
} else {
    System.out.println("Login failed");
}

1️⃣0️⃣ Nested else-if ladder

int age = 20;
String country = "India";

if (age >= 18) {
    if (country.equals("India")) {
        System.out.println("Eligible to vote in India");
    } else {
        System.out.println("Not eligible in this country");
    }
} else {
    System.out.println("Underage");
}

1️⃣1️⃣ else-if vs switch

Else-if ladder is best for ranges and complex conditions, while switch is best for fixed values.

1️⃣2️⃣ Performance Tip

Keep the most common condition at the top to improve performance.
