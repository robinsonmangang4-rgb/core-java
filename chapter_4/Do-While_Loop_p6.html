✅ 1️⃣ Definition of do-while Loop.

A while loop loop is an exit-controlled loop.

That means :

  1. first code executes
  2. then condition is checked
  3. if condition is true → loop continues
  4. if false → loop stops

This is the main difference from while.

Important Point

The do-while loop always runs at least once.

Because condition is checked after execution.
