1️⃣ What is an Operator?

An Operator is a symbol that performs an operation on one or more operands ( values or variables ).

Example 
 Java
 int  c = a + b ;
  • + → operator
  • a,b → operands

◻ Types of Operators in Java

  • Arithmetic Operators
  • Relational Operators
  • Logical Operators
  • Bitwise Operator
  • Ternary Operator
  • Assignment Operator

2️⃣Arithmetic Operators

Used to perform mathematical operations.

Operator Meaning
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus ( Remainder )

Eaxmple Program

 Java
 class ArithmeticDemo {
     public static void main ( String [] args )
        {
          int a  = 10, b = 3;
            
          System.out.println ("Addition: " + (a + b));
          System.out.println ("Subtraction: " + (a - b));
          System.out.println ("Multiplication: " + (a * b));
          System.out.println ("Division: " + (a / b));
          System.out.println ("Modulus: " + (a % b));
                                   
                 }
            }

Output


          Addition: 13 
          Subtraction: 7 
          Multiplication: 30 
          Division: 3 
          Modulus: 1

◻ Note: / gives quotient, % gives Remainder

3️⃣ Relational Operators

Used to compare values.

Result is always boolean ( true / false ).

Operator Meaning
> Greater than
<< /td> Less than
>= Greater than or equal
<=< /td> Less than or equal
= = Equal to
! = Not equal

Eaxmple

 Java
 class RelationalDemo {
     public static void main ( String [] args )
        {
          int a  = 10, b = 20;
            
          System.out.println (a > b);
          System.out.println (a < b);
          System.out.println (a == b);
          System.out.println (a != b);                                   
                 }
            }

Output

4️⃣ Logical Operators

Used with boolean values.

Operator Name Meaning
&& Logical AND true if both are true
| | Logical OR true if one true
! Logical NOT reverse the result

Eaxmple

 Java
 class LogicalDemo {
     public static void main ( String [] args )
        {
          int a  = 10, b = 20, c =  30 ;
            
          System.out.println (a < b && b < c );
          System.out.println (a < b || b < c );
          System.out.println (a > b);
                 }
            }

Output


 true 
 true 
 true
