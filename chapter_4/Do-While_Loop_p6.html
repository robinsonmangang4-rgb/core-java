✅ 1️⃣ Definition of do-while Loop.

A while loop loop is an exit-controlled loop.

That means :

  1. first code executes
  2. then condition is checked
  3. if condition is true → loop continues
  4. if false → loop stops

This is the main difference from while.

Important Point

The do-while loop always runs at least once.

Because condition is checked after execution.

2️⃣ Syntax of do-while .

do{
    //code block
}while(condition);

Structure Explanation 

do{

This starts the loop block.

// code block

this code runs first.

} while(condition);

Condition is checked after execution.

Semicolon ; is mandatory in do-while.

This is very important.

3️⃣ Flow of Execution

          Start
            ↓
       Execute code
            ↓
      Check condition
            ↓
       True → repeat
       False → stop

Do-while Loop diagram

4️⃣ Basic Example 

public class DoWhileExample{
    public static void main(String [] args){
        int i = 1;
        
        do{
            System.out.println(i);
            i++;
        } while(i <= 5);
    }
}

Output 

1
2
3
4
5

5️⃣ Step-by-Step Explanation

Initial value:

int i = 1;

Loop starts:

do  {

Prints value:

System.out.println(i);

Increment:

i++;

Condition:

while (i <= 5);

This continues until i becomes 6 .

6️⃣ How It Works Internally?

Iteration 1:

i = 1 → print 1
i = 2
check 2 <= 5 → true

Iteration 2:

i = 2 → print 2
i = 3
check 3 <= 5 → true

Show on .....

Iteration 5:

i = 5 → print 5
i = 6
check 6 <= 5 → false
stop.

7️⃣ Difference Between while and do-while.

Feature While do-Loop
Condition check Before loop After loop
Minimum execution 0 times 1 time
Type Entry-controlled Exit-controlled

Example Showing Difference

while 
int i = 10;

while( i < 5 ){
    System.out.println(i);
}

Output:

    No Output

Because condition is false initially.

do-while 
int i = 10;

do{
    System.out.println(i);
} while(i < 5);

Output: 

    10

Because it runs once first.

8️⃣ Real-Life Example.

Suppose ATM menu should appear at least once.

int choice = 1;
do{
    System.out.println("ATM Menu");
    System.out.println("1. Withdraw");
    System.out.println("2. Deposit");
    choice--;
}while (choice > 0);

Output: 

ATM Menu
1. Withdraw
2. Deposit
