1️⃣ What is Java?

Java is a programming language used to create desktop applications, web applications, mobile apps (Android), and enterprise software.

3️⃣ History of Java

  • Developed by James Gosling
  • Released In 1995
  • Originally by Sun Microsystems
  • Now owend by Oracle

3️⃣ Features of Java

  • Simple & Easy to Learn
  • Object Oriented
  • Platform Independent
  • Secure
  • Robust & Scalable

4️⃣ Platform Independent

Java follows the principle: "Write Once, Rune Anywhere"

Java program run on any operating system because of the JVM.

5️⃣ Java Architecture 

                .java file
                    ↓
                Java Compiler
                    ↓
                .class file(Bytecode)
                    ↓
                JVM (Java Virtual Machine)
                    ↓
                  Output

6️⃣ JDK JRE & JVM

  • JVM: Executes Java Bytecode
  • JRE: JVM + Libraries
  • JDK: JRE + Compiler + Tools

Relation: JDK → JRE → JVM

7️⃣ First Java Program

             class Hello
              {
                public static void main(String[] args)
                 {
                  System.out.println("Hello World");
                 }      
              }

Explaination of each Line

class Hello

Class name must match the file name.
File name: Hello.java

public static void main(String[]arge)

This is the entry point of the Java program.
JVM starts execution from here.

System.out.println("Hello World");

Prints output on the screen.
println means print and move to the next line.

Program Explanation

  • class Hello → Class Name
  • main() → Program execution starts here
  • System.out.println() → Prints output

8️⃣ How to Compile & Run Program

Step 1: Compile

javac Hello.java

Step 2: Run

java Hello

📌 No .class extension is used while running the program.
