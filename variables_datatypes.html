1️⃣ What is Variable?

A variable is a container used to store data in memory. 

int age = 25;
  • int → data type
  • age → variable name
  • 25 → value stored in memory

📌 Meaning:A box named age stored the value 25

2️⃣ Why Variables are Needeed?

Without variables:

  • You can't store Values
  • You can't perform calculations
  • You can't resuse data
  • Accept user input

Example:

         java 
int a = 10;
int b = 20;
int sum = a+b;
System.out.println(sum);
