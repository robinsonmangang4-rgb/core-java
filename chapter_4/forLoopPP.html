✅ Basic Level ( 1-15 )

1️⃣ Print numbers from 1 to 10 

for (int i = 1; i <= 10; i++){
    System.out.println(i);
}

Explanation : Loop starts at and run until 10 inclusive.

2️⃣ Print numbers from 10 to 1

for (int i = 10; i >= 1; i--){
    System.out.println(i);
}

Explanation : Decrement loop to reverse the order.

3️⃣ Print even numbers from 1 to 20.

for (int i = 2; i <= 20; i+=2){
    System.out.println(i);
}

Explanation : Start at 2 and increment by 2.

4️⃣ Print old numbers from 1 to 20.

for (int i = 1; i <= 20; i+=2){
    System.out.println(i);
}

Explanation : Start at 1 and increment by 2.

5️⃣ Print first 10 natural numbers

Explanation : Natural numbers begin from 1.

6️⃣ Print squares of numbers 1-10

    System.out.println(i * i);

Explanation : Multiply number by itself.

7️⃣ Print cubes of numbers 1-10

for (int i = 1; i <= 10; i++){

✅ Intermediate Level (16-35)

1️6️⃣ Print fibonacci Series (10 terms) 

int a = 0, b = 1;
for (int i = 1; i <= 10; i++) {
    System.out.print(a + " ");
    int next = a + b;
    a = b;
    b = next;
}

Explanation : Each term is sum of previous two.
