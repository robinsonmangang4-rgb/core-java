1. What is a Unary Operator?

A unary operator works on only one operand (one variable/value).

int a = 5;
-a;

2.Types of Unary Operators

Operator Name Example
+ Unary Plus +a
- Unary Minus -a
++ Increment ++a, a++
-- Decrement --a, a--
! Logical NOT !true

3. Unary Plus (+)

Indicates positive value. Mostly optional.

int a = 10;
int b = +a;
System.out.println(b); // 10

4. Unary Minus (-)

int a = 10;
int b = -a;
System.out.println(b); // -10

5. Increment Operator (++)

int a = 5;
int b = ++a;
System.out.println(a); // 6
System.out.println(b); // 6
            
int a = 5;
int b = a++;
System.out.println(a); // 6
System.out.println(b); // 5

6. Decrement Operator (--)

int a = 5;
int b = --a;
System.out.println(a); // 4
System.out.println(b); // 4

7. Logical NOT (!)

boolean isLoggedIn = false;
if (!isLoggedIn) {
    System.out.println("Please login first");
}

8. Tricky Expressions

int a = 5;
int b = ++a + a++;
System.out.println(a); // 7
System.out.println(b); // 12

9. Practice Programs

int a = 10;
System.out.println(a++);
System.out.println(++a);
System.out.println(a--);
System.out.println(--a);

10. Common Mistakes

  • Confusing pre and post increment
  • Writing complex expressions
  • Using ++ inside method arguments

11. Interview Tips

  • Unary operators work on one operand
  • ++ and -- are not atomic
  • Java does not allow ++ on constants
