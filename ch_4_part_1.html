IF and IF-ELSE Statements – Full Course (Theory + Examples)

1. Introduction

In programming, decision-making allows a program to choose different paths based on conditions. The most fundamental decision-making statements are:

  • if
  • if-else
  • if-else-if (else-if ladder)
  • Nested if-else

These statements evaluate conditions (expressions that return true or false) and execute code accordingly.

2. What is a Condition?

A condition is a logical expression that results in either true or false.

Examples of conditions:

  • a > b
  • x == 10
  • marks >= 40
  • isLoggedIn == true

Conditions are formed using relational operators and logical operators.

Common Relational Operators

  • > (greater than)
  • < (less than)
  • >= (greater than or equal to)
  • <= (less than or equal to)
  • == (equal to)
  • != (not equal to)

3. The if Statement

The if statement executes a block of code only if the given condition is true.

Syntax

if (condition) {
    // code executes if condition is true
}

How it Works

  1. The condition inside parentheses is evaluated.
  2. If the condition is true, the code inside the block runs.
  3. If the condition is false, the block is skipped.

Example 1: Simple if

int age = 20;

if (age >= 18) {
    System.out.println("You are eligible to vote");
}

Example 2: if with a false condition

int number = 5;

if (number > 10) {
    System.out.println("Number is greater than 10");
}

4. The if-else Statement

The if-else statement provides two paths:

  • One when the condition is true
  • Another when the condition is false

Syntax

if (condition) {
    // executes if condition is true
} else {
    // executes if condition is false
}

Example: Pass or Fail

int marks = 35;

if (marks >= 40) {
    System.out.println("Pass");
} else {
    System.out.println("Fail");
}

Example: Even or Odd

int num = 7;

if (num % 2 == 0) {
    System.out.println("Even number");
} else {
    System.out.println("Odd number");
}

5. The if-else-if Ladder

Used when multiple conditions need to be checked in sequence.

Syntax

if (condition1) {
    // code
} else if (condition2) {
    // code
} else if (condition3) {
    // code
} else {
    // default code
}

Example: Grade System

int marks = 82;

if (marks >= 90) {
    System.out.println("Grade A");
} else if (marks >= 75) {
    System.out.println("Grade B");
} else if (marks >= 60) {
    System.out.println("Grade C");
} else {
    System.out.println("Fail");
}

6. Nested if-else

An if-else inside another if-else is called nested if-else.

int age = 22;
boolean hasID = true;

if (age >= 18) {
    if (hasID) {
        System.out.println("Entry allowed");
    } else {
        System.out.println("ID required");
    }
} else {
    System.out.println("Underage");
}

7. Logical Operators with if

  • && (AND) – all conditions must be true
  • || (OR) – at least one condition must be true
  • ! (NOT) – reverses the condition
int marks = 70;
int attendance = 80;

if (marks >= 40 && attendance >= 75) {
    System.out.println("Eligible for exam");
}

8. Common Mistakes

  • Using = instead of ==
  • Missing braces { }
  • Incorrect condition order in else-if ladder
  • Writing complex conditions without parentheses

9. Real-Life Examples

int balance = 5000;
int withdraw = 2000;

if (withdraw <= balance) {
    System.out.println("Withdrawal successful");
} else {
    System.out.println("Insufficient balance");
}

10. Summary

  • if checks a condition
  • if-else provides two choices
  • else-if handles multiple conditions
  • Conditions use relational and logical operators
  • if-else is the foundation of program logic
