1️⃣ What is an Operator?

An Operator is a symbol that performs an operation on one or more operands ( values or variables ).

Example 
 Java
 int  c = a + b ;
  • + → operator
  • a,b → operands

◻ Types of Operators in Java

  • Arithmetic Operators
  • Relational Operators
  • Logical Operators
  • Bitwise Operator
  • Ternary Operator
  • Assignment Operator

2️⃣Arithmetic Operators

Used to perform mathematical operations.

Operator Meaning
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus ( Remainder )

Eaxmple Program

 Java
 class ArithmeticDemo {
     public static void main ( String [] args )
        {
          int a  = 10, b = 3;
            
          System.out.println ("Addition: " + (a + b));
          System.out.println ("Subtraction: " + (a - b));
          System.out.println ("Multiplication: " + (a * b));
          System.out.println ("Division: " + (a / b));
          System.out.println ("Modulus: " + (a % b));
                                   
                 }
            }

Output


          Addition: 13 
          Subtraction: 7 
          Multiplication: 30 
          Division: 3 
          Modulus: 1

◻ Note: / gives quotient, % gives Remainder

3️⃣ Relational Operators

Used to compare values.

Result is always boolean ( true / false ).

Operator Meaning
> Greater than
<< /td> Less than
>= Greater than or equal
<=< /td> Less than or equal
= = Equal to
! = Not equal

Eaxmple

 Java
 class RelationalDemo {
     public static void main ( String [] args )
        {
          int a  = 10, b = 20;
            
          System.out.println (a > b);
          System.out.println (a < b);
          System.out.println (a == b);
          System.out.println (a != b);                                   
                 }
            }

Output


 false 
 true 
 false
 true

4️⃣ Logical Operators

Used with boolean values.

Operator Name Meaning
&& Logical AND true if both are true
| | Logical OR true if one true
! Logical NOT reverse the result

Eaxmple

 Java
 class LogicalDemo {
     public static void main ( String [] args )
        {
          int a  = 10, b = 20, c =  30 ;
            
          System.out.println (a < b && b < c );
          System.out.println (a < b || b < c );
          System.out.println (a > b);
                 }
            }

Output


 true 
 true 
 true

5️⃣ Bitwise Operators

Operate on binary (bit-level) data.

Operator Meaning
& Bitwise AND
| Bitwise OR
^ Bitwise XOR
~ Bitwise Complement
<<< /td> Left shift
>> Right shift

Eaxmple

 Java
 class BitwiseDemo {
     public static void main ( String [] args )
        {
          int a  = 5, b = 3;
            
          System.out.println (a & b);
          System.out.println (a | b);
          System.out.println (a ^ b);
          System.out.println (a << 1);
          System.out.println (a >> 1);                                   
                                   
                 }
            }

Explanation

  • 5 → 0101
  • 3 → 0011

Output


 1 
 7 
 6
 10
 2

6️⃣ Ternary Operators

Shortest form of if-else

Syntax

java
condition ? value1 : value2 ;

Eaxmple

 Java
 class TernaryDemo {
     public static void main ( String [] args )
        {
          int a  = 10, b = 20;
            
          int max =(a > b) ? (a : b);
          System.out.println ("Max is: " + max) ;
                                   
                 }
            }

Output


 Max  is: 20

7️⃣ Assignment Operators

Used to assign values.

Operator Example Meaning
= a = 10 assign
+= a+= 5 a = a+5
-= a-= 5 a = a-5
*= a *= 2 a = a*2
/= a /= 2 a = a/2

Eaxmple

 Java
 class AssignmentDemo {
     public static void main ( String [] args )
        {
          int a  = 10 ;
             a += 5 ;
          System.out.println (a);
          a *= 2
          System.out.println (a);                               
                 }
            }

Explanation

  • a += 5; → (a = 10): a += 5: a = a + 5 ( 10+5 = 15 )
  • a *= 2; → (a=15): a *= 2 → a = a * 2 ( 15 * 2 = 30)

Output


 15 
 30
