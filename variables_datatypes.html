1️⃣ What is Variable?

A variable is a container used to store data in memory. 

int age = 25;
  • int → data type
  • age → variable name
  • 25 → value stored in memory

📌 Meaning:A box named age stored the value 25

2️⃣ Why Variables are Needeed?

Without variables:

  • You can't store Values
  • You can't perform calculations
  • You can't resuse data
  • Accept user input

Example:

         java 
int a = 10;
int b = 20;
int sum = a+b;
System.out.println(sum);

4️⃣ Waht is a Data type?

A data type defines:

  • What type of data avariable can store
  • How much memory is allocted
  • What operations are allowed

5️⃣ Types of Data Types in Java

Java has 2 main categories :

Data Types ↓
           ↓→→ primitive Data Types(8 types)
           ↓
           →→→ Non-Primitive Data Types

6️⃣ Primitive Data Types( Very Important )

Java has 8 primitive data types.

Data Type size Example
byte 1 byte byte b = 10;
short 2 bytes short s = 1000;
int 4 bytes int a = 100;
long 8 bytes long l = 992879L;
float 4 bytes float f = 10.5f;
double 8 bytes double d = 99.99;
char 2 bytes char c = 'A';
boolean 1 bytes boolean   isJavaFun = true;

7️⃣ Integer Data Types (byte, short, int, long)

◻ int ( MOST USED )

 java
 int age = 30 ;
 double  salary = 50000 ;

📌 Why int ?

  • Fast
  • Enough of most values
  • Default choice for numbers

◻ Long

Used when numbers are very large. 

 java 
 long mobileNumber = 9876625371L;

⚠ L is mandatory.

8️⃣ Floating Point Data Types ( float & double )

◻ float

 Java
 float percentage = 85.5f;

⚠ f is mandatory

◻ double ( MOST USED for decimals )

 Java
 double price =99.99;

📌 Preferred becouse :

  • more accurate
  • Default for decimal numbers

9️⃣ char Data Type

Stores a single character

 Java
 char grade = 'A';
 char gender = 'M';

⚠Use single quotes ' ' ; not double quotes.

1️⃣0️⃣ boolean Data Type

Stores only true or false.

 Java
 boolean isJavaEasy = true;
 boolean isLoggedIn = false;

📌 Used heavily in :

  • Conditions
  • Loops
  • Login systems

1️⃣1️⃣ Non-Primitive Data Types

These store References, not actual values.

Examples :
  • Sting
  • Arrays
  • Classes
  • Interfaces

◻ String ( VERY IMPORTANT )

 Java
 String name = "Robinson";

📌 Difference :

  • char→ single character
  • String → multiple characters
 Java
 char ch = 'A';       
 String word = "Java";

1️⃣2️⃣ Default Values of Variables

Data Type Default Value
double 0.0
char one character
boolean false/true
string null

⚠ Only applies to instance variables, not local variables.

1️⃣3️⃣ Type Casting ( Bonus Concept )

◻ Implicit Casting ( Automatic )

 Java
 int a = 10 ;
 double b = a;

◻ Explicit Casting ( Manual )

 Java
 double x = 10.5;
 int y = (int) x;

📌 Output : 10 ( decimal removed )
